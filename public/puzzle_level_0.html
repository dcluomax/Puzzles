Single-byte XOR cipher

While using fiddler to debug graph, we found this strange communication is being transmitted. The message looks like some hex encoded key:

0b700b4d0b620b790b780b7f0b540b5f0b6a0b780b7f0b6e0b540b640b6d0b540b5d0b620b680b7f0b640b790b720b76

Out of curiosity, we searched the repo and realize this has been XOR'd against a single character.
However, the key is fetched from KV. Can you find the character, and decrypt the message?

You can do this by hand. But don't: write code to do it for you!
